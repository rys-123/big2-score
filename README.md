# ♠️ 锄大地/大老二 手机记分器 (PWA)

一个简洁、好用、无需下载安装的锄大地（大老二）手机网页记分器。支持罚分自动计算，支持离线使用。

## ✨ 功能特点

*   **📱 像APP一样使用**：支持 PWA，添加到手机主屏幕后，体验和原生 APP 一致，无浏览器地址栏干扰。
*   **📴 离线可用**：在有网时打开一次，之后即使在没网的飞机上、地下室也能正常打开使用。
*   **🧮 自动算分**：
    *   只需输入剩余张数，自动根据规则（<8张1倍，<10张2倍，<13张3倍，13张4倍）计算罚分。
    *   支持“2”的加罚规则（每张2加10分），并自动校验输入合法性。
*   **🚨 输赢判定**：
    *   自动统计总分。
    *   分数 ≥100 时自动标红，提示“爆机”输家。
    *   当局优胜者（0分）显示皇冠 👑。
*   **💾 数据安全**：数据保存在手机本地，刷新页面、关闭浏览器数据不丢失。

## 🚀 如何使用 (在线链接)

**👉 [点击这里直接开始使用](https://rys-123.github.io/big2-score-pwa/)**

> **💡 小贴士**：这是一个 **PWA (渐进式网络应用)**。虽然它是网页，但可以像原生 App 一样安装到手机上，**安装后没有浏览器地址栏遮挡，且支持离线使用（没网也能记分）！**

### 📲 详细安装教程

请根据你的手机型号，按以下步骤操作：

#### 🍎 iPhone (iOS 用户)
1.  **打开链接**：务必使用自带的 **Safari 浏览器** 打开上面的链接。
2.  **等待加载**：打开后建议**停留 10-20 秒**（让系统自动下载离线缓存文件）。
3.  **点击分享**：点击屏幕底部中间的 **分享按钮** (图标是一个方框带个向上箭头 <img src="https://help.apple.com/assets/65E21671239920042405626F/65E21672621C81452D070D18/zh_CN/a4448555078518f886f74a00164627d2.png" width="16"/>)。
4.  **添加到桌面**：在弹出的菜单中向下滑动，找到并点击 **"添加到主屏幕"** (Add to Home Screen)。
5.  **完成**：点击右上角的 **"添加"**。现在桌面上就会出现“锄大地”的图标了！

#### 🤖 Android (安卓用户)
1.  **打开链接**：推荐使用 **Chrome 浏览器** 打开上面的链接。
2.  **打开菜单**：点击浏览器右上角的 **三个点图标 (⋮)**。
3.  **安装应用**：在菜单中找到并点击 **"安装应用"** 或 **"添加到主屏幕"**。
4.  **确认**：根据提示点击确认，图标即会自动出现在桌面上。

---

### 🕹️ 操作指南

1.  **开始记分**：点击底部的 **"📝 记一局"** 按钮。
2.  **输入牌数**：
    *   依次输入每位玩家的剩余张数（数字键盘自动跳过0）。
    *   **特殊规则**：如果开启了"2"的加罚规则，且输入张数 > 0，下方会出现圆圈数字 `1 2 3 4`，请点击选择该玩家手里有几张 2。
3.  **查看结果**：
    *   **输家发红包**：当有人总分 **≥100** 时，名字会变红并显示 💸 图标，顶部会出现红色横幅提示谁需要发红包。
    *   **查看明细**：向下滑动可以看到每一局的详细得分记录。
4.  **数据保存**：
    *   刷新页面数据不会丢失。
    *   如需重新开始（保留名字），点击底部的 **"新的一盘"**。
    *   如需彻底清空（包括名字），点击 **"重置数据"**。

## 📏 计分规则 (罚分制)

1.  **基础倍率**：
    *   剩余 < 8 张：1倍 (n分)
    *   8 ≤ 剩余 < 10 张：2倍 (2n分)
    *   10 ≤ 剩余 < 13 张：3倍 (3n分)
    *   剩余 13 张：4倍 (52分)
2.  **特殊规则**：
    *   当一家出完牌时，其余各家手中每有一张 2，罚分 **+10分**。
3.  **输赢**：
    *   总分 ≥ 100 分为输家（需发红包）。
